MakCraft.Behaviors.Interfaces Namespace
MakCraft.Behaviors.Interfaces 名前空間には、ViewModel と Behavior との間のインターフェイスが含まれます。
Interfaces
  InterfaceDescription
IDialogTransferContainer
生成元ウィンドウからのデータの受取用プロパティのインターフェイス。
IMessageDialogActionParameter
MessageDialogAction へ渡すパラメーターのインターフェイス。 IsDialog が false のときには Button の設定は反映されません。
IViewModelStatus
画面遷移を行うビューモデルの処理状況のプロパティのインターフェイス。
